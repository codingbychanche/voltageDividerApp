Voltage divider App in Google Play besuchen.

This App facilitates the calculation of some properties of resistors. It can be checked if ones calculated resistor value is included in any of the standard series E3 - E96 within a given error margin. Two resistor voltage dividers can be calculated when input and output voltages are known.

Please note: The standard values for the resistors are calculated. This leads to miner errors between calculated and available resistors. Usually this error is not bigger than 1/100.
